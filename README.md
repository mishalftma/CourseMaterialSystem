# Course Material System

**Name:** Mishal Fatima
**Course:** CSC241 – Object Oriented Programming
**Lab Assignment:** 03

## Description

This project is a Java program based on Object Oriented Programming concepts. It manages different types of course materials such as Lecture Notes, Assignment Sheets, and Quiz Files.

Each material has common properties like topic and file size, but each type works differently.

## OOP Concepts Used

* Abstract Class
* Inheritance
* Method Overriding
* Encapsulation
* Constructor Chaining (super)
* this keyword
* Upcasting & Downcasting
* instanceof
* Array of parent references
* toString() method

## Files in Project

* CourseMaterial.java
* LectureNotes.java
* AssignmentSheet.java
* QuizFile.java
